제공 IP 카탈로그 (예시)

PLL (Phase-Locked Loop)

고성능 클럭 생성 및 주파수 합성 IP

ADC (Analog-to-Digital Converter)

정확한 아날로그-디지털 변환 IP

DAC (Digital-to-Analog Converter)

고해상도 디지털-아날로그 변환 IP

LDO (Low Dropout Regulator)

안정적인 전원 공급을 위한 저전압 드롭아웃 레귤레이터 IP

Memory IP
